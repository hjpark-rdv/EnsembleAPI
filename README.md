# Ensemble API

* * *
## Compile
1. Get source code
<pre><code>$git clone https://github.com/rendezvue/EnsembleAPI.git<br>
$cd EnsembleAPI
</code></pre>
2. Submodule update
<pre><code>$git submodule init<br>
$git submodule update</code></pre>
3. make build directory
<pre><code>$mkdif build<br>
$cd build</code></pre>
4. make
<pre><code>$cmake ..<br>
$make</code></pre>
